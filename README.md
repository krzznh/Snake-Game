# Snake-Game
🐍 Snake Game in C
Built a simple terminal-based snake game using C.

Features:
- W,A,S,D controls
- Food system 
- Score tracking
- Increasing difficulty (by reducing sleep time)
- - Real-time movement
- Score tracking
- Collision detection
  
 What I Learned :
- Loops and control flow
- Game logic implementation
- Handling user input

 How to Run  :
1. Compile the code:
   gcc snake.c -o snake
2. Run:
   ./snake
